Esqueleto HTML + Bootstrap (archivo de inicio)

A continuación tienes un ejemplo de un archivo index.html con estructura básica: navbar, login modal y grid de cards. Usa este punto de partida para implementar tu frontend con Bootstrap y enlazar con tu backend PHP.

<!-- index.html (Esqueleto) -->
<!doctype html>
<html lang="es">
<head>
  <meta charset="utf-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1" />
  <title>MoodPlanned</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
      <a class="navbar-brand" href="#">MoodPlanned</a>
      <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navmenu">
        <span class="navbar-toggler-icon"></span>
      </button>
      <div class="collapse navbar-collapse" id="navmenu">
        <ul class="navbar-nav ms-auto">
          <li class="nav-item"><a class="nav-link" href="dashboard.php">Inicio</a></li>
          <li class="nav-item"><a class="nav-link" href="community.php">Comunidad</a></li>
          <li class="nav-item"><a class="nav-link" href="profile.php">Perfil</a></li>
        </ul>
      </div>
    </div>
  </nav>

  <main class="container my-4">
    <div class="mb-4">
      <h2>¿Cómo te sientes hoy?</h2>
      <div class="d-flex gap-2">
        <button class="btn btn-light">😊</button>
        <button class="btn btn-light">😢</button>
        <button class="btn btn-light">😡</button>
        <button class="btn btn-light">😴</button>
      </div>
    </div>

    <div class="row g-3" id="plans-grid">
      <div class="col-6">
        <div class="card">
          <img src="/path/to/img.jpg" class="card-img-top" alt="...">
          <div class="card-body">
            <h5 class="card-title">Plan Título</h5>
            <p class="card-text">Descripción breve...</p>
            <a href="plan.php?id=1" class="btn btn-primary">Ver plan</a>
          </div>
        </div>
      </div>
      <!-- Repetir cards dinámicamente con PHP -->
    </div>
  </main>

  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Cómo usar: abre este archivo en el navegador para ver los wireframes low-fi y copia el bloque de código HTML para empezar el esqueleto. Si quieres, más adelante puedo separar el proyecto en archivos individuales (index.html, dashboard.php, styles.css, script.js).
